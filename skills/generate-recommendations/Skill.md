# Generate Recommendations

Use the Claude Config Manager dashboard's recommendation engine to discover new plugins and MCP servers tailored to your workflow.

To generate recommendations:
1. Open the dashboard: `claude-config start --dev`
2. Navigate to the "Recommended" page
3. Click "Refresh" to generate AI-powered recommendations

Or use the MCP tool: `ccm_open_dashboard` and navigate to Recommended.
